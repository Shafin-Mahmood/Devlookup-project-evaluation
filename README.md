## Devlookup Project evaluation

React + Vite recreation of the  homepage with responsive desktop/mobile layout and hamburger menu behavior.


The goal of this task was to replicate the original website as closely as possible in terms of:

* Design
* Layout
* Spacing
* Typography
* Responsiveness
* Mobile behavior (including hamburger menu)


## 🛠️ Tech Stack

*  React (Vite)
*  CSS3 (Custom styling)
*  Responsive Design (Mobile-first approach)
*  External Assets (Images & Fonts via direct links)

---

## ✨ Features

###  Homepage Recreation

* Fully recreated homepage layout based on the live website
* Maintains visual hierarchy and section flow
* Large typography and modern UI design

### 📱 Responsive Design

* Works seamlessly on:

  * Mobile 
  * Tablet 
  * Laptop 
  * Desktop 
* Adaptive layouts using Flexbox and responsive units

###  Mobile Navigation

* Fully functional **hamburger menu**
* Smooth open/close behavior
* Matches mobile UX of the original website

###  Hero Section

* Video-integrated hero section
* Proper scaling and responsiveness
* Overlay text and call-to-action styling

###  Section-Based Layout

Includes:

* Hero Section
* About / Intro
* Expertise / Services Blocks
* Visual Content Sections
* Footer

###  UI & Styling

* Rounded cards and soft UI elements
* Proper spacing and padding
* Clean color palette matching the original site
* Consistent typography scaling

###  Performance Optimized
* Lightweight Vite setup
* Optimized rendering
* External assets used instead of heavy local files
